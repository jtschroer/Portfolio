## Projects

### Project 1: Sales Dashboard
- **Description**: This interactive sales dashboard provides insights into regional sales performance, product performance, and trends over time.
- **Key Features**:
  - Filter by region, product category, and time period.
  - Visuals: Bar charts, line charts, pie charts, and KPIs.
  - Insights: Identified top-performing regions and sales trends.
- **Tools Used**: Power BI Desktop, Excel, SQL Server.
- **PBIX File**: [Sales_Dashboard.pbix](path/to/sales_dashboard.pbix)

### Project 2: Financial Report
- **Description**: A detailed financial report showing income, expenses, and profits across multiple departments.
- **Key Features**:
  - Drill-throughs to detailed department views.
  - DAX measures for year-over-year analysis.
  - Insights into cost-cutting opportunities and profit optimization.
- **Tools Used**: Power BI Desktop, Excel, Financial Data.
- **PBIX File**: [Financial_Report.pbix](path/to/financial_report.pbix)
- **Screenshot**: ![Financial Report Screenshot](path/to/screenshot.png)

### Project 3: Marketing Analytics Dashboard
- **Description**: A dashboard tracking key marketing metrics such as ROI, customer engagement, and conversion rates.
- **Key Features**:
  - Interactive filters for campaign type and time period.
  - Visuals: Funnel charts, bar charts, and geo-mapping.
  - Insights: Revealed high-performing marketing channels and campaigns.
- **Tools Used**: Power BI Desktop, Google Analytics Data, Marketing Data.
- **PBIX File**: [Marketing_Analytics_Dashboard.pbix](path/to/marketing_dashboard.pbix)
- **Screenshot**: ![Marketing Dashboard Screenshot](path/to/screenshot.png)
